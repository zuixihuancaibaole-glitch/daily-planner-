# 日計劃 · Daily Planner

一個美觀、完全離線的每日任務規劃網頁應用 ✨  
純 HTML + CSS + JS 開發，數據保存在瀏覽器本地，無需註冊、無需伺服器。

## ✨ 主要功能
- 6 種精美主題（暖墨 / 深海藍 / 森林綠 / 午夜黑 / 玫瑰粉 / 沙漠金）
- 時間段任務 + 自動「進行中」高亮
- 時間軸視圖（Timeline）
- 可拖拽排序 + 打印功能
- 便簽（Sticky Notes）
- 每日筆記（自動保存）
- 優先級、分類、進度條統計
- 響應式設計 + 手機友好

## 🚀 快速使用
1. 直接點擊這個連結：https://zuixihuancaibaole-glitch.github.io/daily-planner- （等 Pages 開啟後就會生效）
2. 或下載 `index.html` 雙擊打開（離線也能用）

## 如何自己部署
1. Fork 或下載本倉庫
2. 把 `index.html` 上傳到任何支援靜態託管的平台（GitHub Pages、Vercel、Netlify 等）
3. 享受你的個人日計劃！
